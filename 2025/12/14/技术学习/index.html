1. setup() 函数

  • setup() 的执行时机和作用。
  • 如何在 setup() 中使用响应式 API。

2. 响应式 API

  • ref()：用于基本类型数据的响应式包装。
  • reactive()：用于对象类型数据的响应式包装。
  • toRefs()：解构 reactive 对象以保持响应式。
